## Output

The [`geoip`](#geoip) transform accepts [`log`][docs.data-model.log] events and allows you to enrich events with geolocation data from the MaxMind GeoIP2 and GeoLite2 city databases.
For example:

```javascript
{
  "geoip": null
}
```
