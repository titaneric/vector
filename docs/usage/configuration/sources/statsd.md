# statsd source

![](../../../.gitbook/assets/statsd-source.svg)


The `statsd` source allows you to ingest [`metric`][metric_event] through the StatsD UDP protocol.

## Example

{% code-tabs %}

  type = "statsd"
  address = "127.0.0.1:8126"
